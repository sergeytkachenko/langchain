


 By default, a new section page in the mobile application contains a single group of fields:
 
 General information
 
 . This group of fields cannot be renamed or deleted.
 



 You can add the rest of columns, column groups and details on the section page manually.
 








 Add columns to page
--------------------------


1. Open the System Designer by clicking
 ![btn_system_designer.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/btn_system_designer.png)
 .
2. Click
 
**Mobile application wizard** 

 in the “System setup” block.
3. Select the workplace to edit and click
 **Open** 
 .
4. Click the
 **Set up sections** 
 button.
5. Select a section in the list and click
 **Page setup** 
 ().
6. Click the
 **New column** 
 button in the column group where you need to add a field (
 [Fig. 1](#XREF_79314_28)
 ).
 





 Fig. 1
 

 Adding fields to a page
 

![scr_mobile_wizard_new_column_page.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_new_column_page.png)
7. Select a column in the
 
 Select column
 
 window and click
 **Select** 
 (
 [Fig. 2](#XREF_37746_230)
 ).
 





 Fig. 2
 

 Selecting a column for adding to page
 

![scr_mobile_wizard_new_column.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_new_column.png)
8. To change the order of the fields on the page, drag and drop them with the left mouse button.
9. Click the
 **Save** 
 button (
 [Fig. 3](#XREF_94778_231)
 ).
 





 Fig. 3
 

 Saving a mobile app section page with the new column
 

![scr_mobile_wizard_with_column.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_with_column.png)
10. Click
 **Save** 
 on the mobile application wizard page.
   

 As a result, the changes will become available for all users.







 Add column group to page
------------------------------



 You can add field groups to sections, including groups that are not available in the primary application. The
 
 General information
 
 group cannot be renamed or deleted, but you can add more fields to it.
 



 To add a field group on the section page:
 


1. Open the mobile wizard and open the section page for editing (perform steps
 **1–5** 
 listed in the “
 [Add columns to page](#XREF_78674)
 ” block).
2. Select the
 **Column group** 
 command in the
 **New** 
 menu (
 [Fig. 1](#XREF_82529_232)
 ).
 





 Fig. 1
 

 Adding a field group
 

![scr_mobile_wizard_new_group_column.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_new_group_column.png)
3. Enter a title and a name for the field group.
 


	1. In the
	 **Caption** 
	 field, specify the name that will display in the mobile app.
	2. The
	 **Name** 
	 field is the internal name of the field group and can only contain Latin letters without special symbols.
4. Save the settings by clicking
 
 OK
 
 . (
 [Fig. 2](#XREF_60437_233)
 ).
 





 Fig. 2
 

 Setting up a field group
 

![scr_mobile_wizard_group_column_new.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_group_column_new.png)
5. A new empty field group will appear on the section page (
 [Fig. 3](#XREF_52640_235)
 ). Add fields to it and save the changes on the page.
 





 Fig. 3
 

 Adding columns to a column group
 

![scr_mobile_wizard_column_group_column.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_pages/scr_mobile_wizard_column_group_column.png)
