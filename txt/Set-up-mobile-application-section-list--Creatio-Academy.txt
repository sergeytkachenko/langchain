


 Records in a mobile app list consist of three data blocks: “caption”, “subheading” and “additional columns” (
 [Fig. 1](#XREF_96676_427)
 ).
 



 The cation displays only one column by default. All the other blocks can display any data as per your settings.
 



 A maximum of 13 lookup columns can be displayed in a list. There are no restrictions for the columns of other types.

 List setup procedure
------------------------


1. Open the System Designer by clicking
 
![btn_system_designer.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_sections/btn_system_designer.png)

 .
2. Click
 
**Mobile application wizard** 

 in the “System setup” block.
3. Select the needed section and click
 **List setup** 
 (
 [Fig. 1](#XREF_17333_146)
 ).
 





 Fig. 1
 

 Setting up a section list
 


![scr_mobile_wizard_section_list.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_sections/scr_mobile_wizard_section_list.png)
4. Set up the columns in the section list for the current workplace (
 [Fig. 2](#XREF_51351_112)
 ).
 





 Fig. 2
 

 Adding columns to a section list
 


![scr_mobile_wizard_section_grid.png](/guides/sites/en/files/documentation/user/en/mobile_app_setup/BPMonlineHelp/mobile_app_setup_sections/scr_mobile_wizard_section_grid.png)


1. Click the
 **New column** 
 button and select a column from the list.
2. You can drag and drop columns to change their positions.
3. Click the
 **Save** 
 button.
 



 As a result, new data will be displayed in the section list (
 [Fig. 3](#XREF_43452_431)
 ).
