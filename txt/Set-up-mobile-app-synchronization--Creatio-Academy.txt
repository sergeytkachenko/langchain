



 This functionality is available for Creatio 8.0.5 and later.
 




 You can specify the objects and records to synchronize with the offline mode of the mobile app. For example, this lets you select the records of existing objects. To do this:
 


1. Set up the app in the Mobile Application Wizard. Learn more in a separate guide:
 [Mobile app setup](https://academy.creatio.com/docs/user/customization_tools/mobile_app_setup) 
 .
2. Click
 ![](/sites/default/files/documentation/sdk/ru/BPMonlineWebSDK/Screenshots/TryIt/scr_system_designer.png)
 to open the System Designer.
3. Open the
 
 System setup
 
 block →
 
 Mobile application wizard
 
 .
4. Open the workplace whose synchronization settings to change, for example,
 
 Main workplace
 
 .
5. Click
 
 Actions
 
 →
 
 Set up synchronization
 
 on the toolbar. This opens the
 
 Mobile synchronization settings
 
 page.
 



 The page contains the list of objects the workplace uses (Fig. 1). For example, sections, lookups, details, etc.
 




 Fig. 1 Opening the
 
 Mobile synchronization settings
 
 page
 

![](/sites/default/files/documentation/sdk/ru/BPMonlineWebSDK/Screenshots/SetUpMobileAppSynchronization/8.0/scr_process_of_opening_the_pages.gif)
6. Select an object whose synchronization settings to change.
7. Change the object synchronization settings. You can turn the synchronization on or off for the entire object or filter records to synchronize (Fig. 2).
 




 Fig. 2 Filter settings for the “Activity” object
 

![](/sites/default/files/documentation/sdk/ru/BPMonlineWebSDK/Screenshots/SetUpMobileAppSynchronization/8.0/scr_filter_settings.png)
8. Click
 
 Save
 
 .
9. Review the synchronized records. You can view the record number of a synchronized object in the
 
 Filtered records count
 
 column or synchronize an app on a mobile device and check.
